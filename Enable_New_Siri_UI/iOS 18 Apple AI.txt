For Cowabunga Lite Mac:
1. Download the file
2. Open it in Cowabunga Lite
3. Connect your device
4. Enable the operation in the `Custom Operations` tab at the top
5. Click apply

How to do it on Cowabunga Lite Windows:
1. Open up Cowabunga Lite
2. Connect Device
3. Make sure SkipSetup is checked
4. In file explorer, go to %APPDATA%/CowabungaLite/Workspace, find the UUID, then go to SkipSetup/ManagedPreferencesDomain/mobile
5. Add the plist file (linked below) to that folder
6. Click apply
